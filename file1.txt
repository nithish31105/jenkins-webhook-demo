# 1. Create MySQL deployment
kubectl create deployment mysql --image=mysql:5.7

# 2. Set MySQL root password
kubectl set env deployment/mysql MYSQL_ROOT_PASSWORD=root123

# 3. Expose MySQL inside cluster
kubectl expose deployment mysql --port=3306 --target-port=3306 --type=ClusterIP

# 4. Create phpMyAdmin deployment
kubectl create deployment phpmyadmin --image=phpmyadmin/phpmyadmin

# 5. Connect phpMyAdmin to MySQL
kubectl set env deployment/phpmyadmin PMA_HOST=mysql PMA_PORT=3306

# 6. Expose phpMyAdmin to browser (NodePort)
kubectl expose deployment phpmyadmin --type=NodePort --port=80

# 7. Find NodePort
kubectl get svc phpmyadmin

# 8. Get Minikube IP (if using minikube)
minikube ip

# 9. OPEN IN BROWSER:
# http://<minikube-ip>:<nodeport>
# Example:
# http://192.168.49.2:30080

 HOW TO CREATE A MAVEN PROJECT IN ECLIPSE


1. Open Eclipse IDE.

2. Go to:
   File → New → Project

3. In the list, select:
   Maven → Maven Project
   Click Next.

4. On the next screen:
   • DO NOT tick "Create a simple project (skip archetype selection)"
   Click Next.

5. Select an archetype:
   org.apache.maven.archetypes → maven-archetype-quickstart
   Click Next.

6. Fill project details:
   Group Id: com.example
   Artifact Id: MyProject
   Version: (leave default)
   Package: com.example.myproject

   Click Finish.

7. Eclipse will create the Maven structure:
   src/main/java
   src/test/java
   pom.xml

8. To add dependencies:
   Open pom.xml and add inside <dependencies> </dependencies>

   Example:
   <dependency>
       <groupId>javax.servlet</groupId>
       <artifactId>javax.servlet-api</artifactId>
       <version>4.0.1</version>
       <scope>provided</scope>
   </dependency>

9. To run your project:
   Right-click project → Run As → Java Application
